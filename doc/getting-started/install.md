# Install

{% hint style="info" %}
Make sure you're using `node >= 18` so `fetch` is available
{% endhint %}

### Default installation <a href="#default-installation" id="default-installation"></a>

### npm <a href="#npm" id="npm"></a>

```
npm i midjourney
```

### yarn

```
yarn add midjourney
```
