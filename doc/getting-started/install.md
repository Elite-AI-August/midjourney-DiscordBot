# Install

{% hint style="info" %}
Make sure you're using `node >= 18` so `fetch` is available
{% endhint %}

&#x20;[![npm version](https://img.shields.io/npm/v/midjourney.svg?maxAge=3600)](https://www.npmjs.com/package/midjourney)

### npm <a href="#npm" id="npm"></a>

```
npm i midjourney
```

### yarn

```
yarn add midjourney
```
